API Documentation

Endpoints

GET /

Returns the main HTML page with customizable parameters.

    Parameters:
    - rows: Number of rows (default: 10)
    - cols: Number of columns (default: 10)
    - questions: Number of questions (default: 20)

GET /data

Returns a list of random anime data with one randomly selected answer.

    Parameters:
    - count: Number of anime records to return (default: 10)

GET /api/animes

Returns a list of all animes in the database.

POST /api/animes

Adds a new anime to the database.

    Body:
    {
      "title": "Anime Title",
      "cover": "Cover URL"
    }

DELETE /api/animes

Deletes an anime from the database by ID or title.

    Body (by ID):
    {
      "id": animeId
    }

    Body (by title):
    {
      "title": "Anime Title"
    }

GET /api/random

Returns a shuffled list of random animes.

    Parameters:
    - count: Number of anime records to return (default: 10)

GET /api/tool

Returns the tools HTML page.

GET /api

Returns the API documentation HTML page.
